# 香港大学参观指南（2025年11月15日）

欢迎计划于**2025年11月15日**来参观**香港大学（HKU）**！以下是为您精心准备的校园游览建议，帮助您高效、深入地体验这所百年名校的魅力。

## 📍 校园位置与交通

- **地址**：香港特别行政区南区薄扶林道
- **主要入口**：靠近“香港大学”地铁站B1出口，步行约5分钟可达本部校园
- **推荐交通方式**：
  - 地铁：港岛线至“香港大学站”
  - 巴士：多条线路途经薄扶林道，如12、13、40、103等

## 🏛 必访景点推荐

### 1. 许士芬地质博物馆
- **地址**：大学道4号，本部校园内
- **开放时间**：周一至周六 9:00–17:00（建议提前官网确认）
- **亮点**：
  - 展出珍贵矿物、化石和岩石标本
  - 互动式地球科学展览，适合所有年龄层
  - 免费入场，是了解自然历史的理想场所

### 2. 本部大楼（Main Building）
- **特色**：建于1912年，红砖拱廊与钟楼极具历史感，为香港法定古迹
- **建议停留**：拍照打卡、感受殖民时期建筑美学

### 3. 大陆庭院（The Main Quad）
- **氛围**：绿树成荫，学生休憩交流的中心地带
- **周边**：可顺道参观图书馆外墙与学生会大楼

### 4. 超级一号宿舍观景台
- **视野**：俯瞰维多利亚港与西环城市景观，日落时分尤为壮丽

## 🍽 校园餐饮推荐

- **学苑餐厅（Student Canteen）**：本地风味快餐，价格亲民
- **University Commons**：多国料理自助餐，环境舒适
- **Starbucks（校内分店）**：适合短暂休息、享用咖啡

## 📷 拍照建议

- **最佳拍摄点**：
  - 本部大楼前台阶（全景构图）
  - 地质博物馆大厅（现代展陈设计）
  - 校园步道两旁的凤凰木（秋季落叶美景）

> 💡 提示：2025年11月中旬天气宜人，平均气温约22°C，建议穿着轻便衣物并携带相机。
